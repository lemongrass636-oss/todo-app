
# コンポーネント仕様書: SettingsPage.tsx

## 1. 概要

アプリケーションの動作設定を行うための画面コンポーネントです。
`Material UI` の `Switch` コンポーネントを使用し、ユーザーが直感的に設定のオン/オフを切り替えられるインターフェースを提供します。

## 2. UIコンポーネント構成

Material UI（MUI）の標準的なフォーム要素を使用しています。

| 要素 | コンポーネント | 説明 |
| --- | --- | --- |
| **コンテナ** | `Paper` | 背景を白地（またはダークモード時はグレー）にし、影をつけて浮き上がらせます。 |
| **タイトル** | `Typography (h5)` | 画面の目的を明確に示す見出しです。 |
| **設定項目** | `FormControlLabel` | `Switch` とテキストラベルを紐付け、クリック領域を広げています。 |

## 3. 設定項目一覧

現在実装されている設定オプションは以下の通りです。

1. **ダークモード**:
* 初期値: `true` (defaultChecked)
* 用途: アプリ全体のテーマカラーを反転させます（※現在はUIのみの実装）。


2. **通知をオンにする**:
* 初期値: `false`
* 用途: プッシュ通知やリマインダーの有効化。



## 4. 今後の拡張予定（メモ）

現在のコードは静的な表示（UIのみ）ですが、実際に機能を動作させるためには以下の実装が必要です。

* **状態管理の導入**: `useState` を使用してスイッチの状態を保持する。
* **グローバルテーマとの連動**: ダークモードの切り替えを `App.tsx` の `ThemeProvider` に伝える。
* **永続化**: `TodoPage` と同様に、設定値を `localStorage` に保存してリロード後も設定を維持する。
